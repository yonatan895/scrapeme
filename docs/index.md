# Welcome to scrapeme

`scrapeme` is an enterprise-grade Selenium automation and scraping tool designed for robustness, observability, and scalability.

*   [Deploying the Application](./deployment.md)

## Features

*   **Declarative Scraping:** Define your scraping logic in a simple YAML file.
*   **Resilient:** Built-in retries, circuit breakers, and rate limiting to handle failures gracefully.
*   **Observable:** Detailed Prometheus metrics and structured JSON logs for easy monitoring and debugging.
*   **Scalable:** Designed for containerized environments with support for Docker and Kubernetes.
*   **Extensible:** Easily extendable with custom logic and integrations.
